⚙️ TROUBLESHOOTING:

Q: Port 3000 already in use?
A: Use different port: PORT=3001 npm start

Q: npm install failed?
A: Make sure Node.js v14+ is installed: node --version

Q: Blank screen?
A: Clear browser cache (Ctrl+Shift+Delete)

Q: Server errors?
A: Check terminal output - will show exact error
